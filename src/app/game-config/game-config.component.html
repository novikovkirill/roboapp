Программа рассчитывает для робота оптимальный маршрут от стартовой до конечной точки

Поле задаётся строкой. Допустимые элементы — S (начальная точка маршрута), T (конечная точка), . (доступная для проезда клетка), # (заблокированная).

Элементы S и T должны быть оязательно и присутствовать в единственном количестве. Поле должно быть квадратным

Изначально робот направлен вперёд. Доступные команды — повернуть налево, повернуть направо, проехать вперёд. Каждая команда тратит одну единицу энергии.
